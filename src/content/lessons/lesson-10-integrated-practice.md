## 1. 왜 "종합 연습"이 필요한가?

Lesson 1부터 9까지 각각의 개념을 배울 때는 "이 부분만" 따로 계산했습니다. 약품금액을 배울 땐 약품금액만, 조제료를 배울 땐 조제료만 계산했죠.

그런데 실제 약국에서 청구서를 만들 때는 그 모든 계산이 **한꺼번에, 정해진 순서대로** 이루어집니다. 순서가 틀리면 중간 결과가 달라지고, 결국 최종 청구액이 틀려집니다.

예를 들어, 본인부담금을 계산하려면 반드시 "요양급여비용총액1"이 먼저 확정되어야 합니다. 총액1이 없으면 본인부담금 계산 자체가 불가능하죠. 이 순서를 아는 것이 이번 레슨의 핵심입니다.

처음엔 10단계가 많아 보여도, 각 단계가 무엇을 하는지 한 번만 이해하면 금방 흐름이 보입니다. 같이 정리해 봅시다.

## 2. 10단계 파이프라인 — 전체 흐름 복습

약제비 계산은 아래 10단계를 순서대로 거칩니다. 이 순서는 절대 바뀌지 않습니다.

| 단계 | 이름 | 하는 일 |
|------|------|---------|
| Step 1 | 입력 검증 | 처방전 데이터가 완전한지 확인, 변수 초기화 |
| Step 2 | 마스터 데이터 로드 | 조제일 기준 수가표, 보험료율 불러오기 |
| Step 3 | 약품 순회 | 약품별 금액 계산 (단가 × 수량), 항목 분류 |
| Step 4 | 투약일수 확정 | 내복/외용/주사 최대 일수 확정 |
| Step 5 | 시간/가산 판정 | 야간·공휴일·소아·가루약 가산 조건 확인 |
| Step 6 | 수가 산정 | Z코드별 조제료 금액 결정 |
| Step 7 | 보험료율 + 본인부담금 | 총액1 계산 → 환자 부담 금액 결정 |
| Step 8 | 3자배분 | 환자·공단·(보훈청) 부담 분리 |
| Step 9 | 후처리 | 특수케이스(보훈 감면, 산정특례 등) 보정 |
| Step 10 | 결과 조립 | 최종 청구서 데이터 완성, 출력 |

이 흐름을 머릿속에 한 줄로 외워두면 도움이 됩니다.

> 💡 **외우는 요령**: "검증 → 로드 → 약품 → 일수 → 가산 → 조제료 → 부담금 → 배분 → 보정 → 완성"

## 3. 완전한 실전 예시 — S01 처방전

이제 실제 처방전으로 10단계를 직접 따라가 보겠습니다. 이 예시는 가장 기본적인 건강보험 처방이므로, 처음 연습하기에 딱 좋습니다.

### 처방전 정보

| 항목 | 내용 |
|------|------|
| 환자 | 홍길동, 40세 |
| 보험 | 건강보험 (C10) |
| 조제일 | 2026-04-15 (수요일, 평일 낮) |
| 처방 | 내복약 1종, 7일분 |
| 약품 | 단가 500원, 1회 1정, 하루 3번 |
| 가산 | 없음 (평일 주간 조제) |

---

### Step 1: 입력 검증

처방전 데이터를 받아 필수 항목이 모두 있는지 확인합니다.

- 약품 목록 비어있지 않음 → 통과
- 보험코드 C10 → 건강보험 처방조제, 서식 H024 확정
- 야간/공휴/소아/가루약 플래그 → 모두 false

> ⚠️ **주의**: 이 단계에서 약품 목록이 비어있거나 보험코드가 잘못되면 계산이 즉시 멈춥니다.

---

### Step 2: 마스터 데이터 로드

조제일(2026-04-15) 기준 최신 수가표와 보험료율을 불러옵니다.

- 점수당 단가: **105.5원** (2026년 기준)
- Z코드 수가표 로드 완료
- C10 본인부담률: **30%**, 65세 이상 정액 기준: 10,000원

---

### Step 3: 약품 순회 — 약품금액 계산

약품 목록을 하나씩 돌면서 금액을 계산합니다.

**계산 과정**

- 단가: 500원
- 1회 투약량: 1정
- 1일 투여 횟수: 3회
- 총 투약 일수: 7일

→ 500원 × 1 × 3 × 7 = **10,500원** (원미만 반올림, 여기선 딱 떨어짐)

이 약품은 내복약(급여)이므로 **01항(일반급여)**으로 분류됩니다.

---

### Step 4: 투약일수 확정

약품이 내복약 1종이므로 내복약 최대 투약일수 = **7일** 확정.

이 일수는 다음 단계(Step 6)에서 조제료 Z코드를 선택할 때 사용됩니다.

---

### Step 5: 시간/가산 판정

- 야간? → 아니오
- 공휴일? → 아니오
- 6세 미만? → 아니오 (40세)
- 가루약? → 아니오
- 토요일? → 아니오

가산 없음. 기본 코드를 그대로 사용합니다.

---

### Step 6: 수가 산정 — 조제료 결정

투약일수 7일, 가산 없음 조건으로 Z코드를 결정합니다.

| Z코드 | 항목 | 금액 |
|-------|------|------|
| Z1000 | 약국관리료 | 790원 |
| Z2000 | 기본조제료 | 1,720원 |
| Z3000 | 복약지도료 | 1,150원 |
| Z4107 | 내복약조제료 (7일분) | 4,320원 |
| Z5000 | 의약품관리료 | 680원 |
| **합계** | **조제료(02항)** | **8,660원** |

> 💡 **팁**: Z4107의 "107"에서 "1"은 내복약, "07"은 7일분을 뜻합니다.

---

### Step 7: 보험료율 적용 + 본인부담금

이제 총액과 환자 부담금을 계산할 차례입니다.

**요양급여비용총액1 계산**

```
01항(약품금액) + 02항(조제료)
= 10,500 + 8,660 = 19,160원

총액1 = trunc10(19,160) = 19,160원
```

> **정리**: trunc10이란 10원 미만을 버리는 절사입니다. 19,160원은 이미 10원 단위라 그대로입니다.

**본인부담금 계산**

```
40세, C10 건강보험 → 일반 정률 30%
본인부담금 = trunc100(19,160 × 0.30)
           = trunc100(5,748)
           = 5,700원
```

> **정리**: trunc100은 100원 미만을 버리는 절사입니다. 5,748원에서 48원을 버려 5,700원이 됩니다.

---

### Step 8: 3자배분

청구액(공단부담금)을 계산합니다.

```
청구액 = 총액1 - 본인부담금
       = 19,160 - 5,700 = 13,460원
```

이 처방은 보훈 해당 없으므로 2자배분(환자 + 공단)으로 완료됩니다.

---

### Step 9: 후처리

산정특례, 보훈 감면 등 특수케이스 확인. 이 처방은 모두 해당 없으므로 Step 8 결과를 그대로 유지합니다.

---

### Step 10: 결과 조립

최종 청구서를 완성합니다.

| 항목 | 금액 |
|------|------|
| 약품금액 (01항) | 10,500원 |
| 조제료 (02항) | 8,660원 |
| **요양급여비용총액1** | **19,160원** |
| 본인일부부담금 | 5,700원 |
| **공단 청구액** | **13,460원** |

검증: 19,160 = 5,700 + 13,460 ✓

① 약품금액 → 조제료 → 총액1 → 본인부담금 → 청구액  
② 총액1 → 약품금액 → 조제료 → 청구액 → 본인부담금

<details>

</details>

**Q2.** 총액1이 19,160원이고 보험코드 C10(30%), 환자 나이가 40세일 때, 본인부담금은?

① 5,748원  ② 5,700원  ③ 5,800원  ④ 6,000원

<details>
<summary>정답 확인</summary>

**정답: ②** — 19,160 × 0.30 = 5,748원이지만, 건강보험 본인부담금은 **100원 미만 절사(trunc100)**를 적용합니다. 5,748에서 48원을 버리면 **5,700원**이 됩니다.

</details>

---

**Q3. (계산형)** 총액1이 22,240원이고 환자가 3세(6세 미만)라면 본인부담금은 얼마인가?  
(6세 미만 약국 적용 부담률 = 성인 30% × 70% = 21%)

① 4,600원  ② 4,700원  ③ 6,600원  ④ 6,700원

<details>
<summary>정답 확인</summary>

**정답: ①** — 22,240 × 0.21 = 4,670.4원. 100원 미만 절사 적용 → **4,600원**.  
6세 미만은 성인 부담률(30%)에서 70%만 부담하므로 21%를 적용합니다. 4,670.4의 70.4원을 버려 4,600원이 됩니다.

</details>

### 도전 3: 의료급여 1종 (D10) — CH11 §3.6 기준 정액 본인부담

지금까지는 건강보험(C10) 환자였습니다. 이번엔 **의료급여 1종(D10)** 처방을 계산해 봅시다. 건보와 가장 큰 차이는 **본인부담 방식**입니다. **CH11 §3.6 공식 검증 시나리오(S06)** 와 동일한 입력입니다.

| 항목 | 내용 |

### 도전 4: 의료급여 1종 면제 — 18세 미만 (CH11 §3.12 + CH05 §12.4)

의료급여 1종 수급권자 중 **8종 면제 대상**에 해당하면 본인부담이 **0원**입니다. 8종은 다음과 같습니다:
1) **18세 미만**, 2) 20세 미만 재학생, 3) 임산부, 4) 가정간호, 5) 선택의료급여기관, 6) 행려/노숙인, 7) 결핵·희귀난치·중증질환, 8) 등록 장애인.

이 도전에서는 **18세 미만 자동 면제** 케이스를 계산해 봅니다. **CH11 §3.12 공식 검증 시나리오(S12)** 입니다.

| 약품 | 단가 500원, 1회 1정, 하루 3회 |

### 도전 5: 보훈위탁 (G20+M10) — 3자배분 실전

마지막으로 가장 복잡한 **보훈** 처방입니다. 환자·공단·**보훈청** 세 주체가 부담을 나눠 갖는 3자배분 구조를 직접 계산해 봅시다. webapp 의 **S17 학습 보조 시나리오** 입니다 (CH11 목록엔 없음).

| 항목 | 내용 |
|------|------|
| 환자 | 박충환, 75세 (국가유공자) |
| 보험 | 보훈위탁 (G20) + 보훈감면 (M10) |
| 조제일 | 평일 낮 |
| 처방 | 내복약 2종, 7일분 |
| 약품1 | 단가 800원, 1회 1정, 하루 3회 (고혈압약) |
| 약품2 | 단가 350원, 1회 1정, 하루 3회 (당뇨약) |
| 가산 | 없음 |

**힌트**: 
- **G20**: 보훈위탁 약국(보훈 환자를 위탁받은 일반 약국)
- **M10**: 전액면제 감면 코드 — 환자 본인부담이 **0원**이 되고, 그 금액을 보훈청이 대신 부담
- 결과: `mpvaPrice` (보훈청 청구액) 필드에 값이 채워지고, `userPrice` = 0

## 🎓 이 레슨에서 배운 것

- 약제비 계산은 Step 1(검증) → Step 10(결과 조립)의 순서를 반드시 지켜야 한다
- 총액1 = trunc10(약품금액 + 조제료), 건강보험 본인부담금 = trunc100(총액1 × 부담률)
- 65세 이상 저액 처방은 정액 특례, 6세 미만은 성인 부담률의 70%
- 야간 + 6세 미만이 겹치면 복합 Z코드(Z2000610 등)를 적용한다
- 청구액 = 총액1 - 본인부담금, 이 항등식은 항상 성립해야 한다
