### S06 — 의료급여 1종 (D10, 55세) `FAIL`

**입력**: D10, 55세, sbrdnType="" (기본), 내복2종 5일

| 항목 | 실제 | 기대 | 비고 |
|------|------|------|------|
| sumInsuDrug | 14,250원 | 14,250원 | PASS |
| userPrice | 500원 | 1,000원 (EDB Mcode) | 차이: -500원 |

**원인**: `ENGINE_BUG`  
`medical-aid.ts`의 `resolveMedicalAidFixAmount()` 함수에서 D10 + sbrdnType=""일 때 Mcode 분기에 진입하지 못함.  
현행 코드 로직: `sbrdnType[0] === 'M'`이어야 Mcode 적용.  
sbrdnType=""이면 sbFirst="" → 'M' 조건 불충족 → fixCost(D10=0)로 fallback → default 500원 적용.  

**기대 동작**: sbrdnType=""이거나 null인 경우(기본 수급자)도 D10이면 Mcode를 적용해야 함.

**수정 제안**:
```typescript
// resolveMedicalAidFixAmount() 수정
if (insuCode === 'D10') {
  // sbrdnType이 B로 시작하면 bcode, 아니면 mcode (기본 포함)
  if (sbFirst === 'B') return rate.bcode > 0 ? rate.bcode : 1500;
  return rate.mcode > 0 ? rate.mcode : 1000;  // 기본값 mcode
}
```

if (insuCategory === 'G' || (opt.bohunCode && insuCategory === 'C')) {
